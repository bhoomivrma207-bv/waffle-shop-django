# Django Web Application with Tailwind CSS

A dynamic web application built using Django framework 
with Tailwind CSS integration for modern UI styling.

The project demonstrates backend routing, template rendering, 
database handling, and frontend styling using Tailwind.

---

## Tech Stack
- Python
- Django
- Tailwind CSS
- SQLite

---

## Features
- Dynamic routing
- Template rendering
- Database integration
- Responsive UI using Tailwind
- Structured MVC architecture

---

## How to Run

1. Install Python dependencies:
   pip install -r requirements.txt

2. Install Node dependencies:
   npm install

3. Run Django server:
   python manage.py runserver
